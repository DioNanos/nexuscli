# Security Policy

## Reporting a Vulnerability

For security disclosures, please email **security@mmmbuto.com**.

Please do not file public GitHub issues for security-related problems.

Acknowledgement target: 72 hours. Status update target: 7 days from acknowledgement.

## Supported Versions

Security fixes target the latest released version published on the `main` branch and the most recent release on npm / GitHub Releases. Older versions are not patched separately.
